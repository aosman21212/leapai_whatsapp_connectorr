LeapAI WhatsApp Connector

Overview

The LeapAI WhatsApp Connector is a comprehensive Odoo module that integrates WhatsApp messaging capabilities directly into your Odoo ERP system. This module enables seamless communication with customers through WhatsApp from various Odoo interfaces including sales orders, invoices, and website portals.

Key Features:
  • WhatsApp integration in systray bar for quick access
  • Direct WhatsApp messaging from sales orders and invoices
  • Floating WhatsApp button on website
  • Email and WhatsApp icons in the system tray
  • Document sharing via WhatsApp Web
  • Portal access URL sharing through WhatsApp

Configuration

This module requires minimal configuration and integrates seamlessly with existing Odoo installations. Simply install the module and configure your WhatsApp business account settings.

Compatibility

Compatible with Odoo 18.0 and later versions. Works with standard Odoo modules including Sales, Accounting, and Website modules.
